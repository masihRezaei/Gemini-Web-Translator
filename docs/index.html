About the Extension

Gemini Web Translator is a powerful Firefox extension that harnesses Google's latest Gemini AI models to deliver high-quality translations directly within your browser. What sets it apart from traditional translation tools is its ability to maintain the original page layout and design while translating the content, providing a seamless browsing experience.

In-Place Translation

Translate entire web pages while preserving the original layout and design. No page restructuring or redirection to external services needed.

Text Selection Translation

Select any text on a webpage and instantly translate it with a convenient popup. Perfect for understanding specific sections without translating the entire page.

YouTube Subtitle Translation

Automatically translate YouTube video subtitles to your preferred language with customizable styling options.

Ask AI

Ask questions about any topic, not just the current webpage, and get intelligent answers powered by Google's Gemini AI models.

Multiple Gemini Models

Choose from different Gemini models including Gemini 1.5 Flash, Gemini 2.0 Flash, and Gemini 2.5 Flash Preview for optimal performance.

Mobile Optimized

Enjoy a mobile-friendly experience with a compact popup overlay that works seamlessly on smartphones and tablets.

Supported Languages

Translate content between multiple languages with special support for RTL languages like Persian.

Persian (Farsi) English French German Spanish Arabic Chinese Russian

How to Use

Quick Installation

  1. Download the extension from GitHub Releases
  2. Open Firefox and go to about:addons
  3. Click the gear icon and select "Install Add-on From File..."
  4. Select the downloaded .xpi file

Setup

  1. Get a Gemini API key from Google AI Studio
  2. Click the extension icon in the address bar
  3. Enter your API key in the settings section
  4. Select your preferred Gemini model
  5. Save your settings

Translation Options

  • Full Page: Click the extension icon and use the Translate tab
  • Selected Text: Highlight text on any webpage and click the translation button
  • YouTube Subtitles: Use the Video Translate tab while watching YouTube
  • Quick Translation: Use the keyboard shortcut (Shift+T by default)
