About the Extension

Gemini Web Translator is a powerful Firefox extension that leverages Google's Gemini AI models to provide high-quality translations directly within your browser. Unlike traditional translation tools, it preserves the original page layout and design while translating the content.

In-Place Translation

Translate entire web pages to different languages while preserving the original page layout. No page restructuring or redirection to external services.

Ask Questions

Ask questions about the content of the current web page and get intelligent answers powered by Google's Gemini AI models.

Multiple Gemini Models

Choose from different Gemini models including Gemini Pro, Gemini 1.0 Pro, Gemini 1.5 Pro, and Gemini 1.5 Flash for optimal performance.

Supported Languages

Translate content between multiple languages with special support for RTL languages like Persian.

Persian (Farsi) English French German Spanish Arabic Chinese Russian

How to Use

  1. Install the extension from GitHub Releases
  2. Enter your Gemini API key (get one from Google AI Studio)
  3. Select your preferred Gemini model
  4. Click the extension icon in the address bar to translate pages or ask questions
