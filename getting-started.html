Getting Started with Gemini Web Translator

Welcome to the Gemini Web Translator Firefox extension! This guide will help you get started with using the extension to translate web pages and ask questions about web content using the Gemini API.

Step 1: Get a Gemini API Key

Before you can use the extension, you need to get a Gemini API key:

  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Navigate to the API keys section
  4. Create a new API key
  5. Copy the API key - you'll need it for the extension
Note: Make sure your API key has access to the Gemini 1.0 Pro model. The extension uses the v1 API endpoint.

Step 2: Set Up the Extension

After installing the extension:

  1. Click on the extension icon in the Firefox toolbar
  2. Paste your Gemini API key in the field at the bottom of the popup
  3. Select the Gemini model you want to use from the dropdown menu:
    • Gemini Pro: The standard Gemini Pro model
    • Gemini 1.0 Pro: The Gemini 1.0 Pro model
    • Gemini 1.5 Pro: The Gemini 1.5 Pro model (more advanced)
    • Gemini 1.5 Flash: A faster version of Gemini 1.5
  4. Click "Save Settings"
  5. You should see a confirmation message that the settings were saved successfully
Important: Make sure your API key has access to the model you select. Different API keys may have access to different models.

Step 3: Translating Web Pages

To translate a web page:

  1. Navigate to the web page you want to translate
  2. Click on the extension icon to open the popup
  3. Make sure you're on the "Translate" tab
  4. Select the target language from the dropdown (e.g., Persian, English, etc.)
  5. Click "Translate Page"
  6. Wait for the translation to complete (you'll see a progress indicator)
  7. The text on the page will be translated in-place, preserving the original layout and structure
  8. To restore the original content, click the "Restore Original" button that appears on the page
New Feature: The extension now translates text directly in-place, preserving the original page layout. This means images, formatting, and page structure remain intact while only the text is translated.

Step 4: Asking Questions About Web Pages

To ask questions about the current web page:

  1. Navigate to the web page you want to ask about
  2. Click on the extension icon to open the popup
  3. Click on the "Ask Question" tab
  4. Type your question in the text area
  5. Click "Ask Gemini"
  6. Wait for the response
  7. The answer will appear below the question

Troubleshooting

If you encounter any issues:

  • Make sure your API key is correct and has been saved
  • Check that you have an active internet connection
  • Try different Gemini models if you get an error about a model not being found
  • Common errors and solutions:
    • "models/X is not found for API version Y": Your API key doesn't have access to the selected model. Try a different model.
    • "API key not valid": Double-check your API key or generate a new one.
    • "Quota exceeded": You've reached your API usage limit. Wait or upgrade your API plan.
  • For very large pages, only a portion of the content may be translated due to API token limits
  • If you see an error message, it may provide specific information about what went wrong

Thank you for using the Gemini Web Translator extension! If you have any questions or feedback, please refer to the README.md file in the extension directory.
